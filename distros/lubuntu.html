Lubuntu

beginner stable general low-end based on: ubuntu

Lubuntu is the lightweight member of the Ubuntu family, using the LXQt desktop environment to keep RAM and CPU usage as low as possible. It's the go-to choice for breathing new life into old or budget hardware, while still giving you access to Ubuntu's full software ecosystem.

Desktop Environment LXQt
Package Manager APT
Latest Release 24.04 LTS
Init System systemd
Support Cycle 3 years (LTS)
Architecture x86_64

Key Features

  • Very low RAM footprint, runs comfortably on ~512MB RAM
  • LXQt desktop designed specifically for low-spec and older hardware
  • Full access to Ubuntu's software repositories
  • Lightweight default apps: Featherpad, PCManFM-Qt, QTerminal
  • Easy upgrade path between Ubuntu LTS releases
  • Familiar taskbar layout, comfortable for Windows users
