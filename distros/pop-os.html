Pop!_OS

beginner stable general gaming high-end based on: ubuntu

Pop!_OS is made by System76, a Linux hardware company. It's built for developers and gamers who want a polished, productive desktop with excellent GPU support. It ships with a GNOME-based desktop featuring auto-tiling, and optionally comes with NVIDIA drivers pre-installed so you don't have to chase them down yourself.

Desktop Environment GNOME (COSMIC Shell)
Package Manager APT / Flatpak
Latest Release 22.04 LTS
Init System systemd
Support Cycle ~5 years (LTS)
Architecture x86_64

Key Features

  • Built-in auto-tiling window manager for productive multitasking
  • Dedicated NVIDIA ISO with proprietary drivers pre-installed
  • Optimized for System76 hardware but works great on any machine
  • Pop Shop app store, no Snap packages, Flatpak-first
  • Full disk encryption offered during installation
  • COSMIC desktop environment in active development as a future upgrade
